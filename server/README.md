# Rest API

An Advanced Level Rest API To Fetch, Store, Update & Delete User Details, Sessions, & Products With User Authentication.

## Technologies :

-   Node JS
-   Express JS
-   Typescript
-   Json Web Tokens
-   MongoDB
-   Mongoose
-   Zod Validation
