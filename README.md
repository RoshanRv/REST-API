# Rest API WebApp

A Webapp to Fetch, Add, Update & Delete Users Details, Session, & Products With Authentication.

![preview](https://user-images.githubusercontent.com/75238302/209446220-981df53b-2b1d-4ae5-9d92-848600da9e72.png)

## Technologies :

### Frontend

-   Next JS
-   TailwindCSS
-   SWR
-   React Hook Forms
-   Typescript
-   Zod Validation

### Backend

-   Node JS
-   Express JS
-   Typescript
-   Json Web Tokens
-   MongoDB
-   Mongoose
-   Zod Validation
