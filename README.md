# Rest API WebApp

A Webapp to Fetch, Add, Update & Delete Users Details, Session, & Products With Authentication.

## Technologies :

### Frontend

-   Next JS
-   TailwindCSS
-   SWR
-   React Hook Forms
-   Typescript
-   Zod Validation

### Backend

-   Node JS
-   Express JS
-   Typescript
-   Json Web Tokens
-   MongoDB
-   Mongoose
-   Zod Validation
